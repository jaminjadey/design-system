# @demo-ds/tokens

Generated design-token package for the demo design system.

## Public Exports

- `@demo-ds/tokens`: token maps, metadata, token names, and `cssVar()`.
- `@demo-ds/tokens/token-names`: type-safe token-name helpers.
- `@demo-ds/tokens/tokens.css`: all `--ds-*` custom properties.
- `@demo-ds/tokens/tokens.light.css`: single-mode light CSS.
- `@demo-ds/tokens/tokens.dark.css`: single-mode dark CSS.
- `@demo-ds/tokens/canonical.json`: canonical token contract.
- `@demo-ds/tokens/metadata.json`: package summary.
- `@demo-ds/tokens/token-docs.json`: Storybook-friendly docs data.
- `@demo-ds/tokens/build-report.json`: generation report.

Do not edit files under `dist` directly. Run `pnpm tokens:build`.
