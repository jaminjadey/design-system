# Using Components

Apps import design-system components from `@demo-ds/components`.

```tsx
import { DemoThemeProvider } from "@demo-ds/mantine-theme";
import { Button, Card, PageHeader } from "@demo-ds/components";
import "@demo-ds/mantine-theme/styles.css";

export function App() {
  return (
    <DemoThemeProvider defaultColorScheme="light">
      <PageHeader title="Project overview" />
      <Card>
        <Button>Save changes</Button>
      </Card>
    </DemoThemeProvider>
  );
}
```

Mantine stays behind the design-system package boundary. Application code should
not import Mantine components directly.
