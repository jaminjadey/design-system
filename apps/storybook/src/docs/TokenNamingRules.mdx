# Token Naming Rules

Canonical token names use lowercase kebab-case path segments joined with dots.

```txt
color.primitive.primary.50
color.semantic.text.default
space.md
radius.md
typography.heading.h1
```

Primitive colours live under `color.primitive.*`.

Semantic colours live under `color.semantic.*` and carry both `light` and
`dark` values.

Spacing lives under `space.*` and produces px dimensions.

Radius lives under `radius.*` and produces px dimensions.

Typography lives under `typography.*` and groups font size, line height, and
font weight into one canonical token.
