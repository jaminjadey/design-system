# Demo Design System

This Storybook documents a demo design-system pipeline. It shows how
Figma-style token JSON becomes canonical tokens, generated CSS variables,
generated TypeScript exports, a Mantine theme, and wrapped React components.

The token values and example content are generic demo data. The point is the
package workflow: one token contract feeds the theme, components, docs, and app.

## Package Flow

- `@demo-ds/tokens` exposes generated token artifacts.
- `@demo-ds/mantine-theme` maps generated tokens into Mantine.
- `@demo-ds/components` exports themed wrappers and component patterns.
- Storybook consumes those public package exports.
