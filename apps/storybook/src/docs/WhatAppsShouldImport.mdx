# What Apps Should Import

App code should import:

```ts
import { DemoThemeProvider } from "@demo-ds/mantine-theme";
import "@demo-ds/mantine-theme/styles.css";
import { Button, Card, PageHeader } from "@demo-ds/components";
import { cssVar } from "@demo-ds/tokens";
```

App code should not import:

```ts
import { Button } from "@mantine/core";
```

Mantine is a rendering dependency behind the design-system API. Keeping it
behind the package boundary makes the component contract portable if the
implementation changes later.

CI enforces this with `pnpm architecture:check`, which validates dependency
direction, public package subpaths, app imports, and public declaration files.
